The model package contains a simple object model of the items that make up the gui.

The object model of this API closely matches what you see on screen (see picture above). These five top level objects are all accessible via the com.github.a2g.core.model.authoredroom.IAmTheMainAPI interface.

Most of the other packages in the Core package describe the user interface. With the exception of the AuthoredRoom package, which is used when authoring a room.
